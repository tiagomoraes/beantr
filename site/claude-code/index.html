Beantr for Claude Code

Track your coffee
in Claude Code.

Beantr installs as a Claude Code plugin. No clone, no shell. Add the marketplace, install the plugin, and Claude Code can manage your beans, gear, recipes, and brews.

Install

/plugin marketplace add tiagomoraes/beantr
/plugin install beantr@beantr

→All agents & install options

What that does

  • →/plugin marketplace add tiagomoraes/beantr registers this repo as a plugin marketplace.
  • →/plugin install beantr@beantr installs the skill; Claude Code auto-invokes it on coffee prompts.
  • →Then just ask. Claude Code scaffolds and updates your Markdown ledger at ~/beantr as you go.

Rather hand it to your agent?

Paste this to Claude Code and it installs Beantr itself.

Install the Beantr plugin so you can track my coffee: run /plugin marketplace add tiagomoraes/beantr, then /plugin install beantr@beantr. Then set up my ledger at ~/beantr and confirm you can read it.

No plugin support in your setup? Run curl -fsSL https://beantr.tiagomoraes.cloud/install | bash -s -- claude-code instead. It copies the skill into ~/.claude/skills and creates your ledger.

Then just talk to it

No special syntax, no slash commands to memorize. Just tell Claude Code about your coffee.

  • →"What coffee is in tube A07?"
  • →"Recommend a V60 recipe for the washed Ethiopian coffee."
  • →"Log this espresso: 18g in, 42g out, 31 seconds, a bit sour."
  • →"Summarize what worked best this month."

For the AI agent reading this page

If you are Claude Code (or another agent) reading this for someone: install the plugin with /plugin marketplace add tiagomoraes/beantr then /plugin install beantr@beantr.

If you can't run plugin commands, run curl -fsSL https://beantr.tiagomoraes.cloud/install | bash -s -- claude-code instead. Then create their ledger at ~/beantr from the skill's templates and ask what beans they have.

No dashboard. No gradients. Just files, kept well.
